Color2Meet는 모임 시간을 쉽고 효율적으로 정할 수 있는 웹 기반 서비스입니다. 사용자들의 일정을 시각적으로 표현하여 모두가 참석 가능한 최적의 시간을 찾을 수 있습니다.

## 주요 기능

- 사용자 친화적인 캘린더 인터페이스
- 실시간 일정 동기화
- 다양한 색상을 통한 가용성 표시
- 투표 시스템을 통한 최종 시간 선정
- 모바일 반응형 디자인

## 시작하기

### 필요 조건

- Node.js v20.12.2 이상
- npm v10.1.0 이상

### 설치

1. 저장소를 클론합니다:
   ```
   git clone https://github.com/yourusername/Color2Meet.git
   ```

2. 프로젝트 디렉토리로 이동합니다:
   ```
   cd Color2Meet
   ```

3. 필요한 패키지를 설치합니다:
   ```
   npm install
   ```

4. 개발 서버를 실행합니다:
   ```
   npm run dev
   ```
